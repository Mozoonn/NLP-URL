# NLP-URL
Built a malicious URL classifier using DistilBERT, trained on 650K+ labeled URLs (benign, phishing, malware, defacement). Achieved 98.89% test accuracy with strong overall performance. Shows that lightweight transformers can reliably detect evolving web threats in real time.

Dataset

This project uses the Malicious URLs Dataset from Kaggle, which contains over 650,000 labeled URLs across four categories.

Dataset link:
https://www.kaggle.com/datasets/sid321axn/malicious-urls-dataset

Note: The dataset is not included in this repository. Please download it directly from Kaggle.
